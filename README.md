# ASL Hand Detection using Python

This project detects American Sign Language (ASL) hand gestures using Machine Learning and Computer Vision.

## Features
- Real-time hand detection
- Live webcam prediction
- Dataset collection
- ML model training

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Scikit-learn

## Project Files
- collect_data.py
- train_model.py
- predict_live.py
- hand_test.py
- dataset.csv
- sign_model.pkl

## Installation

```bash
pip install opencv-python mediapipe scikit-learn
```

## Run the Project

```bash
python predict_live.py
```
